SPOTIFY ASSISTANT

GETTING STARTED

1. AUTHORIZATION

Spotify assistant needs to be granted permissions to perform it's actions

Simply navigate to the authentication url

You'll be shown all the actions the bot is allowed to perform on your spotify account. Give it a read and click agree.

You will be presented with the following credentials: clientId, publicKey, privatekey. Note, your public and private Keys are hashed, you can only view them after sign up

2. TRACK SUGGESTIONS

Once sign up is completed, a playlist is created for you titled "S:A Suggestions"

This playlist will be updated every week by scanning your spotify listening activity and suggesting tracks using last fm's API

3. TOP ARTISTS AND TRACKS

To pull listening statistics from, you'll need a GET request to the top-items endpoint with your public key as an authorization header

        
          

            curl
            {{baseUrl}}/{{clientId}}/top-items?
          

          

            -H
            "X-API-KEY: {{publicKey}}"
          

          

            -H
            "Content-Type: application/json"
          

          

            -X
            GET
Parameters
itemType type of item to be retrieved = ["tracks", "artists"]
timeRange time range to retrieve items from ["medium_term", "short_term", "long_term"]
limit optional number of items to retrieve, maximum is 50 and minimum is 1
offset optional number of items to skip

4. SPOTIFY PROFILE

To pull your spotify profile from the API, send a GET request to the profile endpoint

        
          

            curl
            {{baseUrl}}/{{clientId}}/profile
          

          

            -H
            "X-API-KEY: {{publicKey}}"
          

          

            -H
            "Content-Type: application/json"
          

          

            -X
            GET
